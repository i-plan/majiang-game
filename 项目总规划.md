# 泉州麻将项目总规划

## 当前架构状态
- 当前运行主版本为 `minigame/` 微信小游戏工程，`project.config.json` 指向 `minigame/`
- `miniprogram/` 仅保留作历史参考与 page 行为对照，不再作为默认开发主线
- 核心规则与运行时主实现位于 `minigame/game/`，小游戏壳层位于 `minigame/src/`
- 项目范围仍为单机演示：1 名玩家 + 3 个 AI，不接后端，不做联机
- 当前默认以微信开发者工具手工验证为主，不维护自动化测试资产

## 项目目标
- 稳定跑通“首页 -> 对局 -> 结算 -> 下一局”完整闭环
- 保持泉州麻将核心流程和当前已实现地方规则可用
- 在不引入后端和复杂构建链的前提下持续提升可玩性与完成度

## 当前已完成能力
- 首页、牌桌页、结果页已接通
- `gameSession + stateMachine + selectors` 主结构已建立
- 已支持 144 张牌、花牌/补花、开金/金牌百搭、庄闲起手张数、流局、吃碰杠胡过、抢杠胡、三金倒、天胡、天听、游金/双游/三游、主结算与三家番差结算
- 首页、牌桌页、结果页关键跳转/重复触发保护已接通
- scene/runtime 主线已可用
