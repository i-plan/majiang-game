# 泉州麻将小程序项目总规划

## 1. 项目目标

基于**原生微信小程序**实现一个可运行的泉州麻将单机演示版本，满足以下目标：

- 直接在项目根目录运行，不使用 `miniprogram/` 子目录
- 不依赖后端，不做联机对战
- 1 名玩家 + 3 个 AI
- 支持泉州麻将核心流程与主要本地规则
- 能在微信开发者工具中稳定跑通“开局 -> 对局 -> 结算 -> 下一局”闭环

## 3. 当前版本定义

当前项目已经不是“纯骨架”阶段，而是进入了**可玩 MVP + 规则细化阶段**。

### 当前可实现的体验
- 首页可选庄底开始游戏
- 玩家可在牌桌页进行出牌与动作响应
- AI 可自动推进对局
- 对局结束后可进入结果页查看结算
- 可继续下一局或返回首页

### 当前仍需重点补强的地方
- 真机/开发者工具手工验证还不够系统
- 部分泉州规则文案来源是截图，仍有少量边界需继续校准
- UI 仍偏功能型，缺少最终可交付质感
- 自动化测试体系还不够成体系

## 12. 明天继续时建议先做什么

如果明天继续推进，建议按这个顺序开始：

1. 在微信开发者工具手工验证所有特殊规则流程
2. 把验证中发现的问题整理成 bug 列表
3. 优先修正规则偏差和页面交互问题
4. 再补测试脚本，固定回归入口
5. 最后再做 UI 和 AI 优化
