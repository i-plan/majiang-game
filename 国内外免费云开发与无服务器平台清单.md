# 国内外免费云开发与无服务器平台清单

> 更新时间：2026-04-03
>
> 说明：本清单基于各平台官方产品页、定价页或试用页整理，仅收录本次核对时能够明确看到“免费计划”“长期免费额度”或“新用户免费/试用”口径的平台。具体额度、是否需要实名认证/绑卡、是否会休眠或超额扣费，请以开通时官方页面为准。

## 1. 快速结论

### 国内优先看
- **微信小程序 / 小游戏 / 前端全栈**：腾讯云 CloudBase
- **uni-app / uni-app x**：uniCloud
- **轻量函数、定时任务、Webhook**：华为云 FunctionGraph
- **先低成本试用标准 FaaS**：百度智能云 CFC、阿里云函数计算 FC

### 国外优先看
- **移动端 / Web 原型最快起步**：Firebase
- **Postgres + Auth + Storage 一体化**：Supabase
- **开源路线 / 可自托管**：Appwrite
- **边缘函数 / 全球低延迟**：Cloudflare Workers
- **Next.js / 前端全栈部署**：Vercel
- **标准公有云 FaaS**：AWS Lambda、Google Cloud Run、Azure Functions

### 3.3 国外选型建议

- **最省事的全栈原型**：`Firebase`
- **最适合 Postgres 技术栈**：`Supabase`
- **开源优先 / 以后想自托管**：`Appwrite`
- **边缘函数优先**：`Cloudflare Workers`
- **前端项目一键上线**：`Vercel`、`Netlify`
- **标准云厂商函数计算**：`AWS Lambda`、`Google Cloud Run`、`Azure Functions`

## 4. 免费类型说明

不同平台对“免费”的定义不同，选型时建议先分清这几类：

1. **永久免费计划**
   - 例如：Firebase Spark、Supabase Free、Appwrite Free、Nhost Starter、Vercel Hobby、Netlify Free
2. **长期免费额度**
   - 例如：Cloudflare Workers 免费层、AWS Lambda 免费额度、Google Cloud Run 免费额度、Azure Functions 免费额度、华为云 FunctionGraph 月度免费调用
3. **新用户限时免费 / 活动试用**
   - 例如：腾讯云 CloudBase 新用户免费体验、百度智能云 CFC 新用户前三个月免费、阿里云函数计算 FC 免费试用

如果是正式项目，建议重点确认以下问题：
- 是否要求实名认证或绑卡
- 超额后是停服还是自动扣费
- 免费项目是否会因不活跃而休眠
- 是否限制地区、网络、数据库容量、函数执行时长

## 5. 简版选型建议

### 个人开发者 / 原型验证
- 国内：`腾讯云 CloudBase`、`uniCloud`、`华为云 FunctionGraph`
- 国外：`Firebase`、`Supabase`、`Cloudflare Workers`

### 前端全栈项目
- 国内：`腾讯云 CloudBase`
- 国外：`Vercel` + `Supabase` 或 `Firebase`

### 轻量 API / Webhook / 定时任务
- 国内：`华为云 FunctionGraph`、`阿里云函数计算 FC`
- 国外：`Cloudflare Workers`、`AWS Lambda`、`Google Cloud Run`

### 需要后续扩展为正式生产系统
- 国内：`腾讯云 CloudBase`、`阿里云函数计算 FC`
- 国外：`AWS Lambda`、`Google Cloud Run`、`Azure Functions`
